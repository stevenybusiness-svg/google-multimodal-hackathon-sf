**DEMO SCRIPT (follow this exactly, total <=4 minutes):**

**Setup before hitting record:**
- Open https://meeting-agent-31043195041.us-central1.run.app in Chrome
- Start Loom recording (screen + camera mode, recording the browser tab)
- Make sure your face is visible in the Loom camera bubble

**Section 1 — The Hook (~30s)**
- Click "Start Listening" in the meeting agent UI
- Wait for the mic indicator to go active
- Speak clearly: "Let's schedule a follow-up with Alex on Friday at 2pm, I'll take a note to review the proposal, and we should update the project doc."
- Watch the Detected Actions panel — 4 action cards should appear within ~5 seconds:
  1. Calendar event created
  2. Task logged (commitment)
  3. Slack message posted
  4. Document revision detected
- Pause briefly to let the viewer absorb the cards

**Section 2 — Facial Sentiment (~30s)**
- Look at the camera with a neutral/positive expression — note the sentiment badge (should show POSITIVE or NEUTRAL)
- Make an obviously negative expression (frown) — badge should update to NEGATIVE or show change
- Say: "The agent can see — facial sentiment from the camera gates action confidence. Negative sentiment flags actions with a warning."

**Section 3 — Architecture Narration (~60s)**
- Have the architecture diagram ready (from architecture.md or a screenshot)
- Walk through: "This is a four-layer multimodal pipeline:"
  - "HEAR — Google Cloud Speech-to-Text streams real-time transcript"
  - "SEE — Cloud Vision detects facial sentiment from camera frames"
  - "UNDERSTAND — Gemini Flash extracts commitments, meetings, agreements, doc revisions"
  - "ACT — Actions fire autonomously to Google Calendar, Slack, and an internal task log"
- Emphasize: "No human gate. Actions fire in real-time as the conversation happens."

**Section 4 — Live Deployment (~15s)**
- Point to the browser address bar showing: https://meeting-agent-31043195041.us-central1.run.app
- Say: "This is running live on Google Cloud Run. Not localhost — a real deployed service."

**Section 5 — Close (~10s)**
- "This is autonomous meeting intelligence. It sees, hears, understands, and acts — all in real-time, with no human intervention required."
